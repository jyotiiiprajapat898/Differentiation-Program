# Differentiation Program

## Description
This is a Python program that calculates the differentiation (derivative) of a polynomial expression.

## Features
- Calculates derivatives of polynomial expressions
- Simple and easy-to-use Python code
- Beginner-friendly project

## Technologies Used
- Python 3

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/jyotiiiprajapat898/Differentiation-Program.git
   ```

2. Open the project folder.

3. Run the program:
   ```bash
   python Differentiation-Program.py
   ```
